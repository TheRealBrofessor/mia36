Our Mission

Why This Project Exists

Cybersecurity can feel intimidating. We break it down into plain language, hands-on exercises, and relatable analogies.

Plain Language

No jargon without explanation. Every concept is introduced with a real-world analogy you already understand.

Hands-On Practice

You learn by doing. Boot a real OS, run real commands, configure real networks — safely, on your own machine.

Step by Step

Seven phases that build on each other. No gaps, no assumed knowledge. Just a clear path from zero to capable.

Made for Humans

Written by someone who remembers what it's like to be confused. Patient, encouraging, and judgment-free.

The Roadmap

Your 7-Phase Journey

Each phase builds confidence and capability. Click any card to dive into that chapter.

1

Digital Confidence

Boot Ubuntu from a USB drive and explore a new operating system without touching your current setup.

"Test-driving a car before buying"
2

Linux Terminal

Navigate the filesystem, run commands, and understand folders from the command line.

"GPS vs paper map"
3

Networking

Routers, switches, ethernet cables, and the TCP/IP protocol that powers everything.

"The postal system"
4

Internet Traffic

Packets, outgoing connections, DNS resolution — how data actually moves across the internet.

"Tracking a package through checkpoints"
5

Web Navigation

HTTPS encryption, phishing awareness, VPNs, and how browsers keep you safe (or don't).

"Checking the lock on a door"
6

Installing Software

Package managers, repositories, permissions — how software gets onto your system safely.

"Shopping list and store shelves"
7

Home Lab & VM

Set up VirtualBox, create snapshots, and build your own safe practice environment — the capstone.

"Your own car to practice in"
